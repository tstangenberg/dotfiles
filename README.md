# dotfiles #

system &amp; dotfiles setup 


## Requirements ##
### OSX: ###
- XCode Command Line Tools installed


## Features ##
### OSX: ###
- [Homebrew][]
- [Git][]
- [Google Chrome][]
- [Alfred][]
- [Vagrant][]


## Installation ##
```
git clone https://github.com/tstangenberg/dotfiles.git
dotfiles/dots.sh
```

During the installation your password is requested to install some of the apps.
